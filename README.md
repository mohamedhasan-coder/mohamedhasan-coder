## <img src="https://raw.githubusercontent.com/Tarikul-Islam-Anik/Animated-Fluent-Emojis/master/Emojis/Objects/Light%20Bulb.png" width="35"/> Development Philosophy

<div align="center">

<table>
<tr>
<td width="33%" align="center">

**🧩 Clean Code**

I write code that's easy to read, easy to test, and easy to hand off — because the next person maintaining it might be future me.

</td>
<td width="33%" align="center">

**⚙️ Strong Fundamentals**

Frameworks change. Data structures, algorithms, and system design principles don't. I invest in the fundamentals first.

</td>
<td width="33%" align="center">

**🚀 Ship & Iterate**

Perfect is the enemy of shipped. I build, get feedback, and refine — rather than chasing an imaginary final version.

</td>
</tr>
</table>

<div align="center">

### 🔍 Stack Deep Dive

</div>

<table>
<tr>
<td width="50%" valign="top">

**Frontend**

I build interfaces with **React** and **Next.js**, styled using **Tailwind CSS** for utility-first design and **Bootstrap** when I need rapid, consistent UI scaffolding. I care about responsive layouts, accessibility, and clean component architecture.

</td>
<td width="50%" valign="top">

**Backend**

On the server side, I work primarily with **Java + Spring Boot** to build RESTful APIs with proper layering — controllers, services, repositories. I also use **Node.js + Express** for lightweight services and rapid prototyping.

</td>
</tr>
<tr>
<td width="50%" valign="top">

**Database**

I model relational data in **MySQL** with normalized schemas and indexing for performance, and reach for **MongoDB** when I need flexible, document-based storage for rapidly evolving data shapes.

</td>
<td width="50%" valign="top">

**Tooling & Workflow**

**Git & GitHub** are central to how I work — feature branches, meaningful commits, and pull requests. I develop primarily in **VS Code** and **IntelliJ IDEA**, on **Linux**, and test APIs with **Postman**.

</td>
</tr>
